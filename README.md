# BigDataBashScripts
Utility Scripts for the CMPS 364 BigData Course

Make sure to read the header comments in all scripts and ensure they know what they do. These scripts are build for my personal use cases and may not be applicable in all cased. In addtion, there is *minimal* error handling, so missue can have unhandled consequences

For those of you new to UNIX, these can be placed in /bin/ or added your enviroment path to prevent having to type the file path. In addition, it will need to have exectuion permissions to run if it doesn't natively have it.
